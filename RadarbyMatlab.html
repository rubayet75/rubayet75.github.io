Make your radar wireless!!

Use Blutooth module(HC-05) to make it wireless

  • Blutooth TX pin -Arduino RX pin
  • Blutooth RX pin -Arduino TX pin
  • Blutooth 5V -Arduino 5v,GND - GND
  • Pair your Blutooth to your PC
  • Identify the COMport and put the COMport number to the MATLAB code as shown
  • Run the MATLAB code
